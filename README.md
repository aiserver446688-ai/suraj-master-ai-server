# Suraj Master AI Server 🤖

ZephyrAI — Powered by Gemini 3 Flash

## Features
- 🌟 Human-like AI with jokes & emojis
- 🌍 Multilingual (Hindi, Urdu, English, Arabic, etc.)
- 🖼️ Image search via Google/Wikipedia
- 💬 Conversation history with SSE streaming

## API Endpoints
- `POST /api/ai/chat` — Chat with ZephyrAI
- `POST /api/ai/image-search` — Search images
